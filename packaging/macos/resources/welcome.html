Welcome to local-logger

This installer will set up local-logger, a powerful logging and proxy tool for Claude Code development.

What is local-logger?

local-logger is a multi-purpose tool that provides:

  • MCP Server - Model Context Protocol server for logging operations in Claude Code
  • Hook Logger - Captures and logs Claude Code tool usage events
  • HTTPS Proxy - Records Claude API traffic for debugging and analysis

All logs are stored in structured JSON format with automatic daily rotation.

What will be installed?

  • /usr/local/bin/local-logger - Main binary
  • /usr/local/bin/local-logger-uninstall.sh - Uninstaller script
  • /Library/LaunchAgents/org.local-logger.proxy.plist - Auto-start service
  • ~/.local-logger/ - Log files and certificates

What happens during installation?

The installer will:

  1. Install the local-logger binary
  2. Generate TLS certificates for HTTPS interception
  3. Install and trust the CA certificate in your system keychain
  4. Set up and start the proxy service

Important Notes

Administrator privileges are required.

This installer will:

  • Generate a self-signed CA certificate for HTTPS interception
  • Install this certificate to your system keychain with root trust
  • Start a local proxy service on port 6969

Privacy & Security: The proxy only runs locally (127.0.0.1) and all logs are stored on your machine. No data is sent externally.
